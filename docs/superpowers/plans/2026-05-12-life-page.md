# Life Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a light homepage sticky-note link to `/life` and build a dedicated non-design interests page.

**Architecture:** Keep content local and simple. The homepage adds one small link note using the existing pinned-note visual system; `/life` is a new App Router page with local section data and a dedicated CSS module for browser-window-style photo strips.

**Tech Stack:** Next.js App Router, React, TypeScript, CSS Modules, existing global styles.

---

## File Structure

- Modify `src/app/page.tsx`: add the low-priority homepage link note to `/life`.
- Modify `src/app/page.module.css`: style and position the homepage life note on desktop and mobile.
- Create `src/app/life/page.tsx`: render the personal-interest page from local data.
- Create `src/app/life/life.module.css`: style the `/life` page sections, browser-window frames, photo placeholders, responsive behavior, and focus states.
